# Release v1.1.0

**Release Date:** 2025-01-23

---

## New Features

### Release Notes Page (PR #3, Issue #2)
View version history and changes directly within the application.

- Accessible from sidebar navigation
- Displays all releases sorted by version (newest first)
- Markdown content rendered as HTML
- Responsive design matching app theme

### Build Number Footer
Footer component displayed on all pages showing the current build number.

- Shows "dev" during local development
- Displays actual build number in production (set via `BUILD_NUMBER` env var)
- Includes link to Release Notes page

### Updated CLAUDE.md
Project documentation updated to accurately reflect the codebase structure and technologies.

---

## Technical Changes

- Added `releaseNotesController.ts` to serve markdown files from `release_notes/` folder
- Added `/api/release-notes` endpoint (public, no auth required)
- Added `Footer.tsx` component with build number display
- Added `ReleaseNotes.tsx` page component
- Updated `vite.config.ts` to inject `__BUILD_NUMBER__` and `__BUILD_DATE__` at compile time
- Updated `docker-compose.yml` build context to include release_notes folder
- Updated `backend/Dockerfile` to copy release_notes into container
