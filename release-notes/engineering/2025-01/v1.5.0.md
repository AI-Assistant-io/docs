# Release v1.5.0

**Release Date:** 2025-01-24

---

## New Features

### Improved Release Notes Display (PR #20, Issue #16)
The Release Notes page now features a cleaner, more user-friendly interface with version selection.

- **Version Selector**: Horizontal tag buttons to switch between versions
- **Single Release View**: Shows one release at a time for easier reading
- **Latest First**: Automatically displays the most recent release on page load
- **Improved Markdown Styling**: Better formatted content with proper headings, lists, and emphasis

---

## Technical Changes

- Updated `ReleaseNotes.tsx` with version selector state management
- Added `release-notes-content` CSS class with themed styles
- Styled markdown elements (h2, h3, ul, li, strong, code) for consistent appearance
- Version buttons show active state with primary color

---

## UI/UX Improvements

- Clean version selector with clear visual distinction for selected version
- Consistent styling that respects light/dark theme
- Better readability with proper spacing and typography
