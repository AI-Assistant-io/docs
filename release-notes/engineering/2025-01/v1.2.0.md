# Release v1.2.0

**Release Date:** 2025-01-24

---

## New Features

### Vision API Client Sync (PR #4, Issue #4)
Import clients directly from Vision API with selective import and customizable variables.

- **Load Companies**: Fetch available companies from Vision API using your Auth0 authentication
- **Selective Import**: Choose which companies to import with checkboxes
- **Editable Fields**: Set Enterprise ID and Store ID for each company before import
- **Duplicate Detection**: Companies already imported show in "Already Exist" section
- **Settings Integration**: New "Vision Sync" tab in Settings page

### Updated Clients Page
- Removed "Add Client" button - clients must now be imported from Vision API
- "Import from Vision" button links to Settings page
- Empty state message updated to guide users to Vision API import

---

## Technical Changes

- Added `visionApiService.ts` with Vision API integration and company sync logic
- Added `visionSyncController.ts` with REST endpoints for test/preview/sync
- Added `/api/vision/test`, `/api/vision/preview`, `/api/vision/sync` endpoints
- Updated `api.ts` frontend service with Vision API methods
- Updated `Settings.tsx` with Vision Sync tab and import UI
- Updated `Clients.tsx` to remove direct client creation
- Uses UUID for client IDs (compatible with database UNIQUEIDENTIFIER columns)
- Pass-through authentication using user's Auth0 token to Vision API
