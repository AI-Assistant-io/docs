# Release v1.2.0

**Release Date:** 2026-01-21

---

## New Features

### Delete Chat (PR #7, Issue #3)
Customers can now delete their chat conversations, with the thread being removed from Azure Foundry.

- **Confirmation required:** A confirmation dialog prevents accidental deletions
- **Fresh start:** After deletion, a new conversation is automatically created
- **Available everywhere:** Delete option added to all channel interfaces

**Where to find it:**
- **Web Chat:** Trash icon in the header (next to expand/fullscreen buttons)
- **WhatsApp:** Three-dot menu → "Delete chat"
- **Email:** Dropdown menu → "Delete chat"
- **SMS:** Channel menu → "Delete chat"

---

## Technical Changes

- Added `DELETE /api/thread/{id}` endpoint for thread deletion
- Added `delete_thread()` method to `AzureFoundryAgentClient` class
- Added delete UI controls to all four channel templates
